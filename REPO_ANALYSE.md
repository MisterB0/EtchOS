## ⚙️ Installation

### Voraussetzungen
- Terminal-basierte Linux Distro ISO-Image
  - Empfohlen: Raspberry Pi OS Lite, Tiny Core, Arch Linux, etc.
- Etcher oder Balena Etcher (oder Rufus für nicht-RPi)
- Speichermedium (SD-Karte oder USB-Stick)

### Installations-Schritte

1. **ISO-Image herunterladen**
   - Terminal-basierte Linux Distro (Raspberry Pi OS Lite, Tiny Core, etc.)

2. **Etcher herunterladen und starten**
   - Für Raspberry Pi: Raspberry Pi Etcher
   - Für andere: Balena Etcher oder Rufus

3. **Speichergerät auswählen**
   - Raspberry Pi: SD-Karte
   - Sonstige: USB-Stick

4. **ISO-Datei auswählen**

5. **Image flashen und durch Menü klicken**

6. **Speichermedium einbinden und starten**
   - Raspberry Pi: SD-Karte einstecken
   - Sonstige:
     - USB-Stick einstecken
     - In BIOS booten
     - USB-Stick als Boot-Verzeichnis wählen
     - Neu starten
     - Setup folgen (für Arch: `archinstall`)
     - Neu starten und USB ausstecken

7. **EtchOS herunterladen**
   - Neueste Release von GitHub

8. **Zum Download-Verzeichnis navigieren**

9. **EtchOS starten**
   ```bash
   python main.py
   ```

10. **Viel Spaß!** 🎉
